Analyse des Documents SEC Avancé

Ce chapitre couvre les documents SEC les plus importants pour ta pratique quotidienne du trading. Pour chaque document, tu apprendras : ce qu'il contient, où le trouver, quelles sections lire en priorité, et comment l'interpréter.

Le S-1S-1 — Prospectus d'IPO

📖 Définition
Le S-1S-1 — Prospectus d'IPO
Document complet déposé à la SEC avant une IPO. Contient toutes les informations sur l'entreprise. est le document qu'une entreprise doit déposer à la SEC avant son IPO.
C'est le document le plus complet et le plus important qu'une entreprise publiera jamais.
Il contient TOUT : la vérité sur les risques, les finances, le management, la structure du capital.

📖 Définition
Le S-3S-3 — Autorisation d'émission supplémentaire
Document autorisant une entreprise cotée à émettre de nouveaux titres. Un S-3 actif = risque de dilution. est le formulaire qui autorise une entreprise DÉJÀ cotée à émettre et vendre des titres supplémentaires.
C'est l'autorisation légale nécessaire pour un ATM program, un PIPE, ou toute nouvelle émission d'actions.
Un S-3S-3 — Autorisation d'émission supplémentaire
Document autorisant une entreprise cotée à émettre de nouveaux titres. Un S-3 actif = risque de dilution. actif signifie que l'entreprise PEUT diluer ses actionnaires à tout moment.

Le 10-K10-K — Rapport annuel complet
Rapport financier annuel certifié. 60-90 jours après la fin de l'année fiscale. 100-500 pages. — Le Rapport Annuel Complet
📖 Définition
Le 10-K10-K — Rapport annuel complet
Rapport financier annuel certifié. 60-90 jours après la fin de l'année fiscale. 100-500 pages. est le rapport financier annuel complet qu'une entreprise cotée doit déposer dans les 60 à 90 jours après la fin de son année fiscale.
C'est le document le plus complet disponible sur une entreprise. Généralement 100 à 500 pages.
Auditeur externe certifie les chiffres — c'est la garantie d'exactitude.

Un NT 10-K10-K — Rapport annuel complet
Rapport financier annuel certifié. 60-90 jours après la fin de l'année fiscale. 100-500 pages. ou NT 10-Q10-Q — Rapport trimestriel
Rapport financier trimestriel. 45 jours après la fin de chaque trimestre. Moins détaillé que le 10-K. est un signal d'alarme sérieux.
Raisons légitimes : comptabilité complexe, changement d'auditeur, acquisition récente.

Règle : si une small cap dépose un NT → vérifier immédiatement les raisons dans le document.
Si la raison est vague ou liée à des 'enquêtes internes' → signal de vente fort.

📖 Définition
Le 8-K8-K — Rapport d'événement courant
Rapport déposé dans les 4 jours après tout événement matériel. Fait bouger les cours en temps réel. est le rapport d'événement courant. Une entreprise doit le déposer dans les 4 jours ouvrables suivant tout événement 'matériel'.
C'est le document qui fait bouger les cours en temps réel — les algorithmes le lisent en millisecondes.
