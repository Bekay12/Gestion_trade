Comment naviguer sur EDGAREDGAR — Base de données SEC
Electronic Data Gathering Analysis Retrieval. Base de données gratuite de la SEC contenant tous les documents légaux des entreprises cotées.
1Aller sur www.sec.gov La page d'accueil de la SEC. Chercher 'EDGAR Full-Text Search' ou 'Company Search'.
▼
2Chercher par ticker ou nom Dans 'Company Search', entrer le ticker (ex : AAPL pour Apple) ou le nom de l'entreprise. Choisir le bon résultat.
▼
3Sélectionner le type de filing Filtrer par type : 10-K (annuel), 10-Q (trimestriel), 4 (insiders), 8-K (événements), etc. Ou laisser 'All' pour tout voir.
▼
4Lire le document Cliquer sur la date du filing pour ouvrir. Les gros documents ont une table des matières. Utiliser Ctrl+F pour chercher des mots-clés précis.
▼
5Utiliser la recherche full-text EDGAR Full-Text Search permet de chercher un mot-clé dans TOUS les documents SEC. Ex : chercher 'ATM program' pour trouver tous les ATM actifs.

Les professionnels de Wall Street lisent les SEC filingSEC filing — Dépôt SEC
Document officiel qu'une entreprise cotée doit soumettre à la SEC selon un calendrier précis.s chaque matin avant l'ouverture. Voici la routine que tu dois construire :

HeureActionPourquoi
6h00-7h00Vérifier les 8-K déposés depuis hier soirLes annonces importantes arrivent souvent après clôture
7h00-8h00Lire les Form 4 des insiders sur les actions de ta watchlistAchat d'insider = signal haussier fort
8h00-8h30Vérifier les NT 10-K/10-Q récentsRetard de dépôt = signal d'alarme potentiel
8h30-9h00Chercher les 424B et S-3 récentsDilution imminente = éviter ou shorter l'action
9h00-9h30Synthèse + construction de la watchlist du jourSeulement des actions avec catalyseurs vérifiés
⚠️ Attention
Ne jamais trader une action dont tu n'as pas vérifié les filings récents.
Un S-3 actif + ATM program = l'entreprise peut diluer à tout moment pendant ton trade.
Un NT 10-K sur une small cap = signal d'alarme sérieux — problème comptable possible.
Un Form 4 avec achat massif d'un CEO = souvent le meilleur signal d'entrée long.
