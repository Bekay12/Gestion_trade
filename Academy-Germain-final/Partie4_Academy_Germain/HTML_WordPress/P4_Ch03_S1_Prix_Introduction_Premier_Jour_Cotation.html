Introduction en Bourse Intermédiaire

Le S-1 est le document le plus important d'une IPO. C'est le seul endroit où toute la vérité sur l'entreprise doit être dite. Apprendre à le lire te donne un avantage énorme sur les autres traders.

Le formulaire S-1 — Le document de vérité
📖 Définition
Le S-1 est le formulaire que toute entreprise doit déposer à la SEC avant son IPO.
Il contient TOUTES les informations sur l'entreprise : financières, stratégiques, risques, structure de propriété.
Il est public, gratuit, accessible sur EDGAREDGAR — EDGAR
Base de données gratuite de la SEC où tous les documents officiels des entreprises cotées sont archivés. (www.sec.gov).
C'est la loi : l'entreprise doit tout dire — y compris les mauvaises nouvelles.

📖 Définition
Le Quiet periodQuiet period — Période de silence
Restriction légale empêchant les banques et dirigeants de faire des déclarations publiques autour d'une IPO. est une période légale pendant laquelle les banques underwriters et les dirigeants NE PEUVENT PAS faire de déclarations publiques sur l'IPO.
Durée : du dépôt du S-1 jusqu'à 40 jours après l'IPO pour les banques.
💡 Bon à savoir
À la fin du Quiet periodQuiet period — Période de silence
Restriction légale empêchant les banques et dirigeants de faire des déclarations publiques autour d'une IPO., les analystes des banques publient leurs premières recommandations.
Ces recommandations sont PRESQUE TOUJOURS positives — les banques ont été payées pour l'IPO, conflit d'intérêts évident.
Ce moment crée souvent un mouvement dans le cours de l'action.

Le premier jour de cotation — Ce qui se passe vraiment
09:30Ouverture officielle de NYSE / NASDAQ Le marché ouvre mais l'action ne commence PAS à trader immédiatement. Le specialist (NYSE) ou market maker (NASDAQ) équilibre les ordres accumulés.
▼
10h-11hPremier cours fixé Si beaucoup d'acheteurs et peu de vendeurs → premier cours bien au-dessus du prix d'IPO. Si peu d'acheteurs → cours peut ouvrir sous le prix d'IPO.
▼
JournéeVolatilité maximale Volume très élevé. Les institutionnels qui ont reçu des actions au prix d'IPO vendent pour profiter de la hausse ('flipping'). Les retail achètent par excitation.
▼
3-6 moisLe vrai niveau de l'action L'euphorie retombe. L'action trouve son niveau basé sur les fondamentaux. Souvent 20-40% sous le pic du premier jour.

Le Greenshoe — Mécanisme de stabilisation
📖 Définition
Le Greenshoe (Option de surallocationGreenshoe option — Option de surallocation
Mécanisme permettant aux banques de vendre jusqu'à 15% d'actions supplémentaires pour stabiliser le cours post-IPO.) permet aux banques de stabiliser le cours après l'IPO.
Les banques peuvent vendre jusqu'à 15% d'actions supplémentaires.
Si le cours baisse : elles rachètent ces actions sur le marché pour soutenir le cours.
Si le cours monte : elles exercent l'option pour émettre réellement ces 15% supplémentaires.
💡 Bon à savoir
Cas 1 — Cours monte après IPO : les banques exercent l'option → 15% d'actions supplémentaires émises → cours stabilisé par l'offre.
Cas 2 — Cours baisse après IPO : les banques rachètent sur le marché → soutient le cours → protection pour les investisseurs.
