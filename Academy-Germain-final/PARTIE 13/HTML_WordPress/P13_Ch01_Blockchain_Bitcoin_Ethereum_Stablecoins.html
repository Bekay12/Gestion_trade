📖 Définition
Une BlockchainBlockchain — Chaîne de blocs
Base de données distribuée et immuable stockant toutes les transactions d'un réseau. est un registre de transactions partagé entre des milliers d'ordinateurs dans le monde.
Chaque 'bloc' contient un groupe de transactions. Les blocs sont chaînés les uns aux autres — impossible de modifier un bloc sans modifier tous ceux qui suivent.
Résultat : personne ne peut tricher, effacer ou modifier une transaction passée. La BlockchainBlockchain — Chaîne de blocs
Base de données distribuée et immuable stockant toutes les transactions d'un réseau. est immuable.
📊 Exemple concret
Analogie simple :
→ Imagine un cahier de comptes que 10 000 personnes ont chacune une copie exacte.
→ Quand tu fais une transaction, tout le monde la note en même temps.
→ Si quelqu'un essaie de modifier sa copie, les 9 999 autres disent 'faux' — la fraude est impossible.
→ C'est ça une BlockchainBlockchain — Chaîne de blocs
Base de données distribuée et immuable stockant toutes les transactions d'un réseau. : un cahier public, partagé, inviolable.

2012 : 25 BTC → 2016 : 12,5 BTC → 2020 : 6,25 BTC → 2024 : 3,125 BTC
Historiquement, chaque HalvingHalving — Halving / Halvage
Réduction de moitié de la récompense des mineurs Bitcoin. Se produit tous les ~4 ans. a été suivi d'un bull market (~12-18 mois après).
